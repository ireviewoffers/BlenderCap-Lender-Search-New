# BlenderCap Lender Search Demo

A self-contained demo for exploring and filtering lender options. The demo is a
static web app, so it does not require a build step or package installation.

## What is included

- Responsive landing page for the BlenderCap lender discovery experience.
- Search, product, amount, and sort controls that update results in real time.
- Curated sample lender data for CRE, SBA, equipment finance, and working
  capital scenarios.

## Run the demo

Open `index.html` directly in a browser, or serve the folder locally:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.
